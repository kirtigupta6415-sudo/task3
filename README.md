# task3
Decision Tree Classifer Project

This project demonstrates the implementation of a Decision Tree Classifier using Python and Scikit-learn. The objective is to understand how decision trees work for classification tasks, including data preprocessing, model training, prediction, evaluation, and visualization.

Key Features
Data loading and preprocessing
Train-test data splitting
Building a Decision Tree Classifier
Model training and prediction
Performance evaluation using accuracy score
Visualization of the decision tree structure
Understanding Gini Impurity and Entropy criteria

Technologies Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib

Learning Outcomes
Understanding supervised machine learning concepts
Learning how decision trees make classification decisions
Evaluating classification model performance
Visualizing and interpreting decision tree structures
Exploring techniques to prevent overfitting
